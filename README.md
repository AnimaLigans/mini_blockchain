# Mini Blockchain

Полнофункциональный блокчейн на Rust с поддержкой транзакций, цифровых подписей и Proof of Work.

## Архитектура

### Block
- `index` - номер блока в цепи
- `timestamp` - время создания
- `transactions` - список транзакций
- `prev_hash` - хэш предыдущего блока (связывает цепь)
- `hash` - хэш текущего блока
- `nonce` - число для Proof of Work

### Transaction
- `from` - адрес отправителя
- `to` - адрес получателя
- `amount` - сумма
- `signature` - цифровая подпись (ECDSA)
- `public_key` - публичный ключ для проверки

### Wallet
- Генерирует пару ключей (приватный/публичный)
- Подписывает транзакции приватным ключом
- Адрес = первые 10 символов публичного ключа

### Blockchain
- Управляет цепью блоков
- MemPool хранит ожидающие транзакции
- Майнит новые блоки через Proof of Work

## Как использовать с друзьями

Параграфы 12-13 (следующий этап) добавят P2P сеть:
- Каждый запускает `cargo run` на своём ПК
- Сервер слушает порт (например, 3000)
- Друзья подключаются по TCP
- Все синхронизируют блокчейн
- Каждый может отправлять транзакции

Сейчас это работает локально. Скоро будет сетевая версия!
